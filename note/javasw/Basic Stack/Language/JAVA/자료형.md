### 자바의 자료형 : 

> 1. 자바의 자료형은 기본 타입과 참조 타입으로 나누어진다.
> 2. 기본 타입은 정수, 실수, 문자, 논리 리터럴을 저장하는 타입이다.
> 3. 참조 타입은 배열, 열거, 클래스, 인터페이스 등 객체의 위치를 참조하는 타입입니다.
> 4. 기본 타입 : byte, short, char, int, long, float, double, boolean
> 5. 참조 타입 : class, interface, ...
