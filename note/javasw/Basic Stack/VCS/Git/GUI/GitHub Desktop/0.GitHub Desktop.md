### GitHub Desktop

- [깃허브데스크탑설치](https://desktop.github.com/)
- sign in to github.com 클릭 (깃허브 인터넷 연결해서 로그인하기)
- Use my GitHub account name and email address 클릭
- 받을 이메일 주소 선택 (네이버 했음)
- clone a repository from the Intermet... (기존에 있는 거 가져오기)
- 가져올 폴더 선택
